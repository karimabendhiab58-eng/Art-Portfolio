 # ArtPortfolio 🎨

Bienvenue sur **ArtPortfolio**, une plateforme web conçue pour connecter les étudiants artistes avec des établissements professionnels.

## 🚀 Présentation du Projet
Ce projet est réalisé dans le cadre de l'année académique **2025-2026**. L'objectif est de permettre aux artistes de mettre en avant leurs œuvres et aux entreprises de découvrir de nouveaux talents.

### Artiste en vedette : Douaa Ben Jha
La galerie actuelle présente les créations de l'artiste **Douaa Ben Jha**, incluant des styles variés comme le Digital Art, le Réalisme et l'Abstrait.

---

## 🛠️ Stack Technique

### Backend (Laravel 11)
- **Framework :** Laravel 11 / PHP 8.2
- **Base de données :** PostgreSQL
- **API :** REST API sécurisée pour la gestion des œuvres.

### Frontend
- **Technologies :** HTML5, CSS3, JavaScript (ES6+)
- **Communication :** Fetch API vers le serveur Laravel.
